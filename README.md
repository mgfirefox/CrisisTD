Проект 3D Tower Defense в Unity (C#)

Использован VContainer как IoC для DI, создания собственного игрового цикла, использования C# интерфейсов и классов (в том числе как entry point сцен).

Использован паттерн MVPS для создания архитектуры логики игровых сущностей. Model является C#-интерфейсом, Presenter и Service – C#-классом, View – MonoBehaviour.

Реализована waypoint-система для создания маршрута врагов. В prefab она представляет собой последовательность линейной, квадратичной и кубической линий Безье, в коде разбивается на последовательность точек маршрута.

Реализованы сущности Enemy и Base. Enemy движется, используя маршрут waypoint-системы, при достижении конца пути наносит урон базе. В свою очередь, Base умирает при потери всего своего здоровья.

Реализована сущность Tower. С помощью паттерна Strategy действия башни разбиты на отдельные классы. Реализованы два действия башни: действие атакующей башни и башни поддержки. Аналогичным образом выделены атакующие действия башни: атака по одиночной цели, атака по площади, атака по углу (в виде дроби дробовика), атака в виде очереди выстрелов; а также действие башни поддержки в виде постоянного баффа к дальности другим.

Реализована сущность Player. Управление реализовано с помощью InputSystem, переключение камеры – с помощью Cinemachine. а интерфейс – с помощью Unity UI.

Реализована механика размещения башни: создание областей запрета на размещение башни у размещенных башен и на протяжении всего маршрута врагов, создание preview выбранной башни, ее области дальности и области запрета размещения башни.

Реализована механика взаимодействия с башней: выбор башни, ее улучшение и продажа, отображение ее характеристик.

Внедрены модели и анимации (с помощью Unity Avatar, Animation Clip, Animation Event, Animator и Animator Controller), звуки (Unity Audio Clip и Audio Source) и системы частиц (Unity Particle System).
